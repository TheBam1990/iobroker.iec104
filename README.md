# ioBroker IEC 60870-5-104 Adapter

Please choose a language:

- [English documentation](README.en.md)
- [Deutsche Dokumentation](README.de.md)

Kurzfassung: Der Adapter kann als IEC-104-Master oder -Slave laufen. Empfangene Punkte werden nach ASDU/Common Address unter `ASDU-<Adresse>` abgelegt. Darunter gibt es getrennte Ordner fuer Werte, IV, NT, Zeitstempel und COT.

Short summary: The adapter can run as IEC-104 master or slave. Received points are grouped by ASDU/common address below `ASDU-<address>`. Each ASDU contains separate folders for values, IV, NT, timestamps and COT.
